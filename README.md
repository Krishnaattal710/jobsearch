# Job Search Project

## Introduction
This project is designed to help users search for job openings efficiently. With an intuitive user interface, users can filter jobs based on various criteria, such as location, industry, and experience level.

## Features
- **User Registration**: Users can create accounts to save their job searches.
- **Job Filtering**: Users can filter job listings based on location, job type, and more.
- **Application Tracking**: Keep track of jobs youâ€™ve applied for.
- **Notifications**: Receive notifications for new jobs that match your criteria.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation
To get started with the project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/Krishnaattal710/jobsearch.git
cd jobsearch
npm install
```

## Usage
To run the application, use the following command:

```bash
node app.js
```

Then open your web browser and navigate to `http://localhost:3000`.
